Patch note #10

This relates development from 2015/06/15 to 2015/06/26.

Applications

Emails (v0.9.4)

  • Desactivate filter when user changes mailbox;
  • Compatibility for oauth2 gmail account;
  • Don't display new messages in filter;
  • Deleting several message should remove them instantly from list;
  • Fix sender/recipient filter exit;
  • Improve conversation management;

Calendar (v1.0.16)

  • Fix alarms;
  • Popover is not closed when month is changer.
  • Fix display of long calendar name;
  • Fix display of all day long events;
  • Renaming calendars with long names was sometimes difficult.

Contacts (v 1.1.5)

  • Sort of contact without first name has been improved
  • Display contact suffixe in list

Files (v1.1.9)

  • You can now select a file by clicking on its icon;

Konnectors (v0.3.1)

  • Fix installation error on instances using the `.deb` package

Platform

Home (v2.0.3)

  • You can now open an application from the application management screen;
  • Installed application list now also shows the branch of the remote repository, allowing to know if your using a stable or development version of the application;
  • Updating an application icon doesn't require anymore to re-install the app;
  • Prevent users of managed instances to break them by altering their url;
  • Starting an application from status panel opens it;
  • When an application has no current logs, display the last log file we have
  • Remove notification list when open an application from it;
  • Fix hard disk usage and RAM calculus that was sometime wrong;
  • Fix a problem that prevented to restart the server;
  • Fix a display bug of application update button;
  • Forbid to set an empty contact address ;
  • Display an error message when contact address is wrong ;
  • Fix translation of messages when restarting the platform;
  • Fix a bug some sometime prevented to remove some notifications;
  • Weather application was still visible in market place after its installation. It's now fixed;
  • Fix a bug that prevented to send message to support when an application was broken;
  • Removing a connected device was broken, it should be fixed now.

Development tools

Cozy-Monitor (v1.2.26)

  • `cozy-monitor status` now has a `--raw` option to use its output in other commands;

Documentation

  • Our user Aleth Gueguen added a section explaining how to synchronize with Cozy an existing calendar in Thunderbird.
