Patch note #8

This relates development from 2015/05/18 to 2015/06/01.

Applications

Emails (---)

  • Fix fields From and To in mail header description.
  • Improve all confirmation dialogs.
  • Really remove draft instead of moving it to trash.
  • Keayboard navigation doesn't start infinite scroll.
  • All messages list is loaded when application is in the background.
  • Canceling a draft creation should not display warning when the draft is empty.
  • Open next message after a deletion.
  • Fix locale.
  • Signature isn't changed if mailbox changes in compose page.
  • Deleting all displayed messages doeesn't load next messages.
  • Speed up infinite scrolll.
  • Don't navigate to next message when cancelling answer.
  • Fix message drag and drop.
  • Add possibility to close a message.
  • Add possibility to display message in full screen.
  • Some lines in messages list are too long.
  • Improve search exit.
  • Add filter on date.
  • Unable to re-open message preview after closing it.

Files (v1.1.8)

  • Server could be closed if the download was too long. (see [#279](https://github.com/cozy/cozy-files/issues/279))
  • Folder name was not trimmed (see [#282](https://github.com/cozy/cozy-files/issues/282))
  • Fix upload folder when the folder already existed.
  • Bulk actions icons have been changed for clarity.
  • Overall improvements of multiple selection handling.
  • Files and folders can now be selected by clicking their row .
  • Multiple elements can be selected by holding shift+click.
  • Minor fixes in navigation during uploading.
  • Performance improvements. Big folders are still hard to browse, we are still working on it.
  • Sharing emails have received a style update.

Photo (v1.0.13)

  • Photos imported from Files could not be downloaded.
