Patch note #2

This relates development from 2015/02/23 to 2015/03/06.

Applications

  • Applications should not use Flash anymore.

Calendar (v1.0.11)

  • Fix calendar name when importing ICS.
  • Fix a bug where invitations lead to an error.
  • There was a rare case where timezone was not taken into account during import.
  • Fix a bug in event creation from the modal.
  • Better defaut calendar (import, new event).

Contacts (v1.1.12)

  • Tags are no longer displayed in the list.
  • Better name form.
  • Better fonts.

Emails (v0.6.9)

  • Fix contact labels switch.
  • Display all contact as label in mail edition.
  • Unable to type a contact name with space.
  • Add automatic mailbox refresh.
  • Improve new line in mail sending.
  • Fix moving conversations from message list.
  • Fix problem of random refreshing of account creation screen.

Files (v1.1.3)

  • Store file last modification date during upload.
  • Release of the desktop client for debian 32 & 64 bit
  • Release of the desktop client for OSX

Photos (v1.0.8)

  • Sharing screen didn't display contacts and their picture.
  • Shared albums could not be downloaded as zip.
  • Uploading numerous photos at the same time broke the application.
  • Rotating a photo was not always persistent.

Mobile application (v0.1.3)

  • Fix button download on files.
  • Increase spped of image synchronization.
  • Increase speed navigation in folders.
  • Improve french translation for synchronization button in menu.
  • Remove pull-to-refresh.
  • Add space between menu button and breadcrumb.
  • Update pouchdb.

Platform

Controller (v2.0.17)

  • Better error handling during platform update.

CozyDB (v0.0.8)

  • Logging is managed by printit.
  • Reduce requests were not taken into account.

Development tools

Documentation

  • Localization of the end-user documentation.
  • Many, many improvements!

cozy-dev (v1.0.21)

  • Remove "halt" option for vm:stop command ("halt" is now the default behaviour).
  • Display URL after "deploy" command (only for user application).
  • Improve update-image function to prevent conflict in vagrant's box management.
  • Improve detection of versions when offline.
  • Fix "deploy" command for home.

VM image (v0.10.0)

  • Install imagemagick (usefull for thumbmails)
  • Upgrade coffee-script
  • Update cozy-stack
