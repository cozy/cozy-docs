Patch note #15

This relates development from 2015/09/25 to 2015/10/18.

Highlights

The main update is of course the new version of the Contact application. It's a full rewrite of the client side of the application, so no changelog this time.

During the last weeks, we also updated all the projects to ease the use of Transifex for localization. If you want to help us translate Cozy, please join our Translator team on Transifex.

We are updating all applications to allow an upgrade to Node 0.12. The middle term goal is to be compatible with the LTS Node 4.2
