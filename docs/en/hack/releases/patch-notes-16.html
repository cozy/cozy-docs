Patch note #16

This relates development from 2015/10/19 to 2015/11/02.

Highlights

Here's another big release for Contacts, with better search, batch actions and the ability to merge two contacts;

We also did a lot of minor improvements and bug fixes in all our applications, the complete list is below.

Applications

Calendar (1.0.28)

  • This version should work better with Node 4.x
  • Contributors: aenario

Contacts (2.1.0)

  • Better search: you can now use advanced search parameters, and the results are sorted;
  • Display is updated on the fly when you add a contact from anoter application, like Emails;
  • You can now merge two contacts, for example if you imported them twice
  • You can select multiple contacts to perform actions on them: delete, export, mergeâ€¦
  • Contributors: aenario, jacquarg, kelukelu, m4dz

Import-from-google (0.1.5)

  • Log client errors on server side

  • Contributors: clochix, frankrousseau, jacquarg, kelukelu
