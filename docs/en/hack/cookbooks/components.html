Components

Here is a list that references the components of Cozy.

The stack

What we call the "stack" are the core components of Cozy:

  • Controller: the PaaS that run the various applications
  • Carapace: the process wrapper for NodeJS the controller uses
  • Data System: unified persistence layer
  • Proxy: handles registration and authentication
  • Home: end-user friendly Cozy manager

The last one is not a true Cozy component but is used to install and manage self-hosted Cozy: cozy-setup.

Applications

Official

These applications are officially maintained by Cozycloud:

  • Files: files and folder management
  • Calendar: easy yet powerful calendar management
  • Contacts: a personal relationships manager
  • Emails: an email client
  • Sync: contact and calendar synchronization

Community contributions

These applications have been released by members of the Community or by Cozycloud (but aren't officially maintained)

  • Bookmarks: manage your browser's bookmarks
  • Cozic: your music whenever you need it
  • Databrowser: browse all your data
  • Feeds: a RSS feed reader
  • Konnectors: bring more data to your Cozy
  • Kresus: an open-source personal finances management app
  • KYou: Know Yourself through various measures
  • MesComptes: manage your banking accounts
  • Nirc: an IRC client
  • Notes: self-explanatory
  • OWM: weather forecast
  • Photos: manage and share your photos
  • Todos: list, prioritize, be more productive

Modules / Libraries

This is a set a of libraries developed and maintained by Cozycloud that are used in Cozy.

Generic

  • request-json: simple JSON-only requests
  • cozy-vcard: a vcard parser
  • cozy-ical: an iCal parser
  • americano: makes Expressjs more opinionated about configuration

Cozy specific

  • americano-cozy: [DEPRECATED] an americano plugin to ease the use of the Data System
  • cozy-realtime-adapter: let your app be notified in realtime of changes
  • cozy-notifications-helper: east notifications for your app
  • cozy-i18n-helper: enable localization in your application with no effort
  • jugglingdb-cozy-adapter: [DEPRECATED] Data System ODM plugin
  • cozydb: Data System ODM plugin
  • cozy-clearance: sharing feature in a snap of a finger

Tools

Using/developping with Cozy is easy thanks to the existing tools:

  • cozy-monitor: easy Cozy management (full documentation)
  • cozy-dev: your best friend to developer Cozy applications
  • cozy-fixtures: add fixtures to the database to test your application
