Installer Cozy sur Pingfiles

Cette documentation concerne l’installation de l’ancien prototype de Cozy, aujourd’hui déprécié et dont nous n’assurons plus le support. La documentation d’installation de la nouvelle version sera disponible très bientôt.

Démarrez rapidement avec Cozy et les serveurs privés virtuels pré-configurés de Pingfiles.

C'est très facile, tout peut être fait depuis le browser.


1. Connectez vous à la page produit VPS de PingFiles.

Lien vers la page VPS : https://www.pingfiles.com/serveur-prive-virtuel.html

Choisissez le format de VPS qui vous convient. Nous recommandons d'utiliser la version Connect au minimum.


2. Configurer votre VPS pour qu'il soit basé sur Cozy.

Pour cela sélectionnez Cozy Cloud - 64 bits dans le champ Système d'exploitation puis cliquez sur Continuer.

3. Procédez au paiement et attendez l'email de confirmation.

C'est bon ! Obtenir un VPS Cozy avec PingFiles est très simple. Vous allez recevoir un email de leur part confirmant que votre VPS a bien été livré avec les instructions pour s'y connecter.

Et maintenant vous pouvez profiter de votre Cozy !
